Jaylub Bot

Discord bot for music, moderation, chaos & voice commands

What is Jaylub?

Jaylub is a feature-packed Discord bot built with discord.js. It supports music streaming, voice chat interaction, moderation tools, AI integration and custom meme commands.

Main Features

• SoundCloud & YouTube music playback
• Voice channel control & streaming
• Message moderation (kick, timeout, delete messages)
• Custom meme responses (jaylub / holub system)
• AI integration via OpenAI API
• Discord RPC + presence customization

Personality

Jaylube is designed to be chaotic, humorous, and aggressive in tone. It responds differently based on user ID, creating personalized interactions for specific users.

Music System

jaylube zahraj [query] → plays SoundCloud/YouTube audio
Supports streaming via discord voice channels
Auto disconnect after track ends

Moderation

jaylube kickni @user → kick member
jaylube vysprchuj @user → timeout user
jaylube smaz [number] → bulk delete messages

Memes & Replies

jaylub → funny reply per user ID
holub → alternative meme response
hardcoded personality interactions

AI Integration

The bot can communicate with OpenAI API for advanced responses, allowing intelligent chat-based interactions inside Discord.

System Architecture

Built using Node.js + discord.js v14
Uses ytdl-core, play-dl, yt-search for media handling
Includes voice system via @discordjs/voice
Supports async streaming & event-driven logic

Admin Controls

Owner-only commands:
• restart bot (329)
• shutdown bot (328)
• restricted execution system
