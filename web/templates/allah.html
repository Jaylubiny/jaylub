Allah Bot

Discord bot for fun, voice, music and AI chat

What is Allah?

Allah is a multi-purpose Discord bot built for entertainment, voice interaction, music playback, and AI chatting. It connects to local AI models and supports Czech language responses.

Core Features

• Text commands
• Voice TTS (speech synthesis)
• YouTube music playback
• AI chat via local LLM (LM Studio)
• Voice channel interaction

AI Personality

The bot uses a custom system prompt that makes it respond in Czech, with short and direct sentences, simulating a local personality style.

Commands

!cigi → replies "CIGI"
napis [text] → sends message
jaylub → sends GIF
328 → owner-only script trigger

Voice

rekni [text] → TTS speech in voice channel
zahraj [query] → plays YouTube audio
smrdcis → disconnect from voice

AI Chat

allahu [prompt] → chat with local AI model (Gemma 3 4B)
Responses are always in Czech language.

System Info

Built with discord.py, yt-dlp, gTTS and aiohttp.
Uses LM Studio API for local LLM inference.
Runs as a lightweight async Python service.
