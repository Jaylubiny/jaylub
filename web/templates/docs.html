About the Website

Jaylub is the public website for the Jaylub project and the starting unregistered company Jaylubiny. It collects landing pages, documentation, authentication-protected tools, chat, and browser games in one Go-powered site.

Frontend

The frontend uses HTML templates rendered by the Go backend. A shared layout provides navigation, authenticated user display, global styling, and static CSS/JavaScript loading. Individual pages add their own template content and lightweight scripts only when needed.

Backend

The backend is a Go HTTP server with route definitions, page handlers, API endpoints, and authentication middleware. Login sessions are stored in SQLite, protected routes read the authenticated website account, and game/chat APIs use the same account identity without separate user systems.

Architecture

Requests enter through Cloudflared, reach the Go server on the Raspberry Pi, pass through routing and authentication, then either render templates or call APIs backed by SQLite.
